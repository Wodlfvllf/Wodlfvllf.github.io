# Shashank Shekhar Shukla — Portfolio

A responsive, dependency-free portfolio for applied ML, research, and systems work.

## Run locally

Open `index.html` in a browser. There is no package installation, server, or build step.

## Host free with GitHub Pages

1. Push these files to a GitHub repository.
2. In **Settings → Pages**, choose **Deploy from a branch**.
3. Select the `main` branch and `/ (root)`, then save.

For the `Wodlfvllf.github.io` repository, the site will be published at https://wodlfvllf.github.io/. Relative asset paths also allow the site to work from a normal project repository.

## Structure

- `index.html` — content and page structure
- `styles.css` — visual system and responsive layouts
- `script.js` — navigation, active sections, and reveal motion
- `favicon.svg` — site mark

All résumé content is directly editable in `index.html`.
